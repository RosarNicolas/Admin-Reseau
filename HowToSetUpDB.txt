-----------------------------
Se connecter au serveur MySQL
-----------------------------

Déployer my sql container 

Récupérer l'image Docker MySQL :
	docker pull mysql/mysql-server:latest

Déployer le container : 
	docker run --name=mysql101 -d mysql/mysql-server:latest

Vérifier que le container tourne bien :
	docker ps

Un mot de passe pour la databse à été créé aléatoirement, récupérer ce mdp :
	docker logs mysql101

Se logger dans le serveur "containerisé" : 
	docker exec -it mysql101 mysql -uroot -p
....et entrer le password récupéré dans les logs (étape précédante)

Modifier le mot de passe :
	ALTER USER 'root'@'localhost' IDENTIFIED BY 'new_password';
//Attention à ne pas oublie le point-virgule !! Nous sommes dans mySQL

Se déconnecter pour tester le nouveau mot de passe :
	exit

Vérifier le mot de passe :
	docker exec -it mysql101 mysql -uroot -p
	-> new_password à entrer

-----------------------------
CREER LA DB 
-----------------------------
Créer la db (en évitant l'erreur si une db du même nom existe déjà) :
	create database if not exists docker_project;

On peut voir la liste des db avec la commande :
	show databases;

Pour voir quelle database est en cours d'utilisation :
	select database();
	-> retourne NULL si aucune db n'est sélectionnée

Selectionner une db :
	use DB_NAME;

Vérifier qu'on est dans la bonne db :
	select database();
	-> retourne le nom de la db sélectionnée

CREER LES TABLES
-----------------------------
mysql> create table users(
    -> NOM varchar(20) not null,
    -> PRENOM varchar(10) not null,
    -> CLASSE char(4) not null,
    -> primary key (NOM,PRENOM,CLASSE));

mysql> create table products(
    -> ID int not null auto_increment,
    -> NOM varchar(40) not null,
    -> DESCRIPTION varchar(100) not null,
    -> PRIX float(8,2) not null,
    -> QUANTITE int not null,
    -> primary key (ID));
